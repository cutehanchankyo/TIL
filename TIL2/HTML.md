# HTML 용어

## HTML개요 중요 용어
- < head > :문서의 각종 정보와 문서 자체에 대한 설명을 담는부분

- < body >:웹브하우저 화면에 실제로 출력해서 보여 주려는 모든 정보 또는 내용이 들어가는 부분

- < title >:문서의 제목을 표시하는 부분 표시

## HTML에서 텍스트 표현

< br >:키드의 엔터키와 같은 줄바꿈을 지정 자유 형식

< hr >:수평선을 그리는 태그

< h1 > ~< h6 >:제목 글자의 크기,진하게 표시되면 자동 줄 바꿈의 적용
