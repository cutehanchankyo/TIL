# 반복문

## while

형식
```java
while(조건){
    반복 실행 영역
}
```

예제
```java
while (true){
    System.outprintln("Coding EveryBody")
}
```

## for

형식
```java
for(초기화; 종료조건, 반복실행){
    반복적으로 실행될 구문
}
```

예제
```java
public static void main(String[] args) {
        for (int i = 0; i < 10; i++) {
            System.out.println("Coding Everybody " + i);
        }
 
    }
```
